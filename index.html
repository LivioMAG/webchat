      Admin-Zugriff & Supabase-Regeln

      Die folgenden Hinweise und SQL-Blöcke koppeln den Vollzugriff an app_profiles.is_admin = true. 

      
              
      

              
      
                
      
                  
      1. Auth
      
                  
      
                    Login erfolgt weiterhin normal über Supabase Auth mit E-Mail und Passwort.
                    Die Visualisierung wird nur angezeigt, wenn das eingeloggte Profil
                    is_admin = true hat.
                  
      
                
      
                
      
                  
      2. Tabellen
      
                  
      
                    Admin-Profile mit is_admin = true erhalten globalen Zugriff auf
                    app_profiles, weekly_reports und
                    holiday_requests.
                  
      
                
      
                
      
                  
      3. Storage
      
                  
      
                    Im Bucket weekly-attachments dürfen Admin-Profile alle Objekte
                    lesen und verwalten; normale Benutzer bleiben auf ihren eigenen Ordner
                    beschränkt.
                  
      
                
      
                
      
                  
      4. Web-Export
      
                  
      
                    Der PDF-Export erstellt ein kombiniertes Dokument pro Kalenderwoche und listet
                    zusätzlich fehlende Rapporte auf.
